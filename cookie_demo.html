Πώς λειτουργεί το Floating Cookie Button

Οδηγίες:

  1. Όταν φορτώσεις τη σελίδα, θα εμφανιστεί το consent banner
  2. Αφού κάνεις οποιαδήποτε επιλογή (Accept/Reject/Customize), θα εμφανιστεί το floating button κάτω αριστερά
  3. Το floating button είναι στρογγυλό και μπλε, με εικονίδιο cookie
  4. Πατώντας το, ανοίγει το preferences modal για να αλλάξεις τις ρυθμίσεις
  5. Σε mobile, εμφανίζεται μόνο το εικονίδιο (χωρίς κείμενο)

Σημείωση:

Το floating button εμφανίζεται μόνο αφού ο χρήστης έχει αλληλεπιδράσει με το consent banner. Αυτό εξασφαλίζει ότι δεν ενοχλεί χρήστες που δεν έχουν κάνει ακόμα επιλογή.

Χαρακτηριστικά του Button

  • Θέση: Κάτω αριστερά της οθόνης (fixed position)
  • Σχήμα: Στρογγυλό (border-radius: 50px)
  • Χρώμα: Μπλε με διαφάνεια (rgba(59, 130, 246, 0.9))
  • Εικονίδιο: SVG cookie icon
  • Κείμενο: "Cookies" (κρύβεται σε mobile)
  • Animation: Smooth fade-in με scale effect στο hover
  • Z-index: 99 (πάνω από άλλα στοιχεία)

CSS Code

#cookieSettingsBtn {

    display: none;

    justify-content: center;
}

#cookieSettingsBtn:hover {
    background-color: rgba(37, 99, 235, 1);
    transform: scale(1.05);

}
